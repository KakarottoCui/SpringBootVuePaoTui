# SpringBootVuePaoTui
 基于java SpringBoot和Vue的校园跑腿帮买系统设计

演示视频链接：https://live.csdn.net/v/286326

实现的功能

两种角色：管理员和用户

管理员：学校管理、用户管理、任务管理、评价管理、公告管理等

用户：注册登录、任务管理（接、发跑腿任务）、订单管理、公告管理、评价管理、个人中心等

使用的技术

后端：JAVA开发语言，SpringBoot框架，MySql数据库，Maven

前端：Vue
